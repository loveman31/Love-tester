# 💕 Love Tester

En morsom og interaktiv nettside hvor du kan teste kompatibiliteten mellom deg og den personen du liker!

## Funksjoner

- 💑 Test kjærlighetskompatibilitet basert på navn
- ✨ Vakker og responsiv design
- 🎨 Smooth animasjoner og overganger
- 📱 Fungerer perfekt på mobil, tablet og desktop
- 🌐 Enkelt å bruke grensesnitt

## Hvordan bruke

1. Åpne `index.html` i nettleseren din
2. Skriv inn ditt navn
3. Skriv inn navnet på personen du liker
4. Klikk på "Test kjærligheten!"
5. Se resultatet ditt! ❤️

## Teknologi

- **HTML5** - Struktur
- **CSS3** - Styling og animasjoner
- **Vanilla JavaScript** - Interaktivitet

## Filer

- `index.html` - Hovedfilen med HTML-struktur
- `style.css` - Styling og layout
- `script.js` - JavaScript-logikk for beregning av score

## Tips for forbedringer

Du kan gjøre nettsiden enda bedre ved å:

- Legge til flere algoritmer for score-beregning
- Dele resultatet på sosiale medier
- Lagre historikk over tidligere tester
- Legge til flere emojis og animasjoner
- Gjøre designet enda mer personlig
